SCUBE is a Flutter-based mobile application designed to monitor and visualize solar power plant performance. The app provides a clear and responsive interface to view power generation data, inverter status, energy statistics and weather-related information. It is built to help users easily understand system performance and make informed decisions.

## Technologies Used
This project is built using Flutter as a cross-platform framework and Dart as the programming language. Google Fonts are used for clean typography, and Flutter SVG is used for scalable vector graphics. The app is structured with custom widgets to maintain a clean UI.

## How to Run the App

Before running the project, ensure that Flutter is properly installed on your system.

First, verify your Flutter installation and environment setup:

    flutter doctor

Install all required dependencies for the project:

    flutter pub get

Check the list of connected devices or available emulators:

    flutter devices

Run the application on the default connected device:

    flutter run

Run the application on a specific device using its device ID:

    flutter run -d <device-id>


### Build the App

Build an Android APK:

    flutter build apk

Build the application for iOS:

    flutter build ios

Build the web version of the application:

    flutter build web

This project was built as a learning focused Flutter application to demonstrate responsive UI design, reusable widget architecture, and dashboard-style interfaces.
